# NewComplementarity-CK-MOOD-Metrics
Dataset for software defect prediction using CK/MOOD and proposed metrics across 17 PROMISE projects, highlighting metric complementarity.

## Projects Included
Ant, JEdit, Log4j, Lucene, POI, Velocity, Xalan, Xerces (multiple versions)

## Research Objective
To empirically demonstrate that:
- the proposed metrics provide additional discriminative power,
- their combination with CK/MOOD metrics improves defect prediction performance.

## Reproducibility
These datasets were used to evaluate multiple machine learning models under 10-fold cross-validation, including statistical validation using:
- Mann-Whitney U test
- Wilcoxon Signed-Rank test
- Confidence intervals

## Availability
The dataset is publicly available to support reproducibility and further research.
